詳解

（※翻訳者追記） いわゆるプログラムで言う所の、インストラクション（Instruction；命令）だ。その抽象クラス

  • （※翻訳者追記）リファクタリング指南
  • （※翻訳者追記）　　わたしがもっといい名前空間を考えてあげよう。　 GatchaGatchaGames.RPGMakerUnite.Engine.Instructions.Abstract クラスだ。その方が適切だと頭の中で思っておけばよい
  • （※翻訳者追記）　　空間は作るものではなく、１つの大きな空間を切っていく（パーティションしていく）感覚だ。作るではなく、切ると表現することにしよう
  • （※翻訳者追記）　　理想的には名前空間 GatchaGatchaGames.RPGMakerUnite.Engine.Instructions を切る
  • （※翻訳者追記）　　理想的にはこのクラス名を単に Abstract とする
  • （※翻訳者追記）　　プログラムで使う時はファイルの冒頭あたりで using TheInstruction = GatchaGatchaGames.RPGMakerUnite.Engine.Instructions; と、名前空間の別名を切る
  • （※翻訳者追記）　　プログラムでこのクラスを使う時は TheInstruction.Abstract のように型指定する
